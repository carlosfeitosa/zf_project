O sistema detectou um conflito na versão dos dados que você esta tentando submeter.

Isto foi causado pelo seguinte motivo: as informações que você esta submetendo foram
alteradas antes de você finalizar sua edição.

Para resolver este impasse, escolha uma das opções abaixo:

Escolha esta opção se desejar visualizar as informações que constam atualmente no banco de dados
apenas. Não haverá perda, nem dados sobrescritos.

Escolha esta opção se desejar visualizar as informações que constam atualmente no banco de dados,
no formulário de submissão. As informações que você tentou submeter serão perdidas. Você podera
alterar as informações e submeter novamente, se assim o desejar.

Escolha esta opção se desejar passar por cima da atualização, sem revisitar nenhum formulário ou
informação salva anteriormente.

Escolha esta opção se desejar cancelar sua submissão. Neste caso, as informacoes por você enviadas
serão ignoradas.
